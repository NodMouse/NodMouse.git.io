IEEE 编码

-9.625

首先-9.625为负数 则标志位为1
然后将-9.625转换为二进制-1001.101
移位小数点移位三位得到1.001101
则8位指数位置为011+0111 1111=1000 0010 移位数字+127
最后求23位尾数去掉小数点左边的1得到001101000000000000000000(23位)懒得数0
得到结果 1100 0001 0001 1010 0000 0000 0000 0000
0X C 1 1 A 0 0 0 0

-8

首先-8位负数 标志位为1
然后-8转换为二进制-1000.000
移位小数点3位得到-1.00000
则8位指数位为 011+0111 1111=1000 0010
尾数为0则得到00000000000000000000000000000000(23位)
最后结果为1100 0001 0000 0000 0000 0000 0000 0000
C 1 0 0 0 0 0 0

8.25

首先8为正数 标志位为0
然后8.25转换为二进制1000.01
移位3位得到1.00001
则8位指数位为011+0111 1111=1000 0010
尾数为0.00001 则23位尾数位为 00001000000000000000000000(23位)
最后结果为 0100 0001 0000 0100 0000 0000 0000 0000
0x 4 1 0 4 0 0 0 0

IEEE的一些特殊取值

1.先把要转换的数字转换为二进制

2.把小数点挪到第一个有效数字的右边

3.把后23位写成小数点右边的数,补够23位

4.第一位数为0是正数为1是负数

5.如果左移小数点,则中间8位第一位是1,然后位移的次数减去1写到中间8位的低位. 如果右移小数点,中间8位第一位是0,然后位移的次数减去1,把负数的二进制数写到低七位.
